# PyLauncher
 Launcher alternatif d'empisurvie pour les radins

# Crédits
Libraries python utilisés:
-Colorama
-Tkinter
-WGET
-PIL
-re
-os
-minecraft launcher lib (https://pypi.org/project/minecraft-launcher-lib/)
